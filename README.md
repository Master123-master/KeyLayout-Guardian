# KeyLayout Guardian 🛡️⌨️

A lightweight Windows background utility designed to lock and enforce your preferred keyboard layout, preventing games and applications from forcefully changing it.

## 🎯 The Problem
Many non-English PC users (such as Hungarian, German, or French layout users) experience a frustrating Windows issue: certain video games like **Roblox**, older MMOs, or Source Engine games (CS:GO / CS2) automatically switch the active system keyboard layout to English (QWERTY) upon launch. 

This disrupts in-game chat, breaks custom hotkeys, and forces players to constantly and manually press `Alt + Shift` or `Win + Space` to fix it while playing.

## 🚀 The Solution
**KeyLayout Guardian** runs silently in your system tray. You select your preferred language layout once, toggle it on, and the program guards it. If any external application or game attempts to change your system's keyboard language, KeyLayout Guardian instantly intercepts the change and forces it back to your choice within milliseconds.

## 🛠️ Features
- **Set & Forget:** One-click activation to lock your layout.
- **Instant Interception:** Reverts unauthorized layout changes in milliseconds.
- **Roblox Fix:** Completely eliminates the automated English layout bug in Roblox.
- **Lightweight:** Negligible CPU and RAM usage while running in the background.

## 📥 Installation & How to Use
1. Go to the **Releases** section on the right side of this page.
2. Download the latest compiled `KeyLayout-Guardian.exe`.
3. Launch the program, select your desired keyboard language from the list, and turn on the protection.
4. Keep it minimized and enjoy uninterrupted gaming!

## ⚙️ Requirements
- Windows 10 / 11
- Python 3.x (Only if you are running it directly from the source code)
